

# Scripts bash

Ensemble de scripts bash qui permettent de lancer l'infra et les services du poc notifications :
- `start-X.sh` pour démarrer un service (3 instances)
- `stop-X.sh` pour arrêter un service
- `restart-X.sh` pour redémarrer un service
- `X-all.sh` pour arrêter, démarrer ou redémarrer tous les services (3 instances)
- `X-minimal` pour arrêter, démarrer ou redémarrer tous les services (1 instance)
- `X-infra` pour arrêter, démarrer ou redémarrer l'infra

Les scripts sont à lancer depuis la racine du projet (par exemple `./scripts-bash/start.all.sh`).
